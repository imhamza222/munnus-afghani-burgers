# Munnu’s Afghani Burger

Responsive React + Tailwind website for Munnu’s Afghani Burger, Bahria Town Phase 4.

## Edit restaurant content

- Restaurant details, contact links, ratings, address, and timings: `src/data/restaurantInfo.js`
- Menu categories, items, prices, and customer favorites: `src/data/menuData.js`
- Main food image: `public/images/munnus-food-spread.jpg`

Use `Ask for Price` for any menu item without a confirmed price.

## Run locally

```bash
pnpm install
pnpm dev
```

## Production build

```bash
pnpm build
```

The deployable output is generated in `dist/`.
